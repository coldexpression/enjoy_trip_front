> [SSAFY] 공공데이터를 이용한 여행지 추천 웹 사이트

😃 공공데이터를 이용하여 여행지 정보를 보기 쉽게 조회할 수 있습니다.

😃 원하는 여행지를 북마크 등록 및 삭제할 수 있습니다.

😃 게시판 기능을 활용하여 다양한 사람들과 소통할 수 있습니다.

😃 Vue 부트스트랩과 CSS 를 활용하여 웹 페이지를 구성하였습니다.

😃 http://localhost:8080 에서 접속이 가능합니다. [ 실행 필수 ]
